Diretrizes LGPD no GoPDF

O GoPDF adota práticas de governança para tratamento de dados pessoais em conformidade com a Lei Geral de Protecao de Dados (Lei nº 13.709/2018).

1. Base legal e finalidade

Os dados sao tratados para execução de contrato, seguranca da conta, faturamento de assinatura e prevenção a fraude.

2. Minimização de dados

  • Coletamos apenas dados necessórios para autenticacao e prestação do serviço.
  • Dados sensíveis de pagamento são processados por parceiro especializado.

3. Direitos do titular

  • Confirmação de tratamento e acesso aos dados.
  • Correção, anonimização e eliminação quando aplicável.
  • Revogação de consentimento quando esta for a base legal.

4. Segurança e resposta a incidentes

Aplicamos controles técnicos e administrativos para reduzir risco de acesso nao autorizado e manter rastreabilidade de eventos críticos.

5. Canal de atendimento

Demandas sobre protecao de dados devem ser enviadas para suporte@gopdf.com.br.
